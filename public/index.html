INSTRUCTIVO DE USO
A — Instalar la app
1. Guarda el archivo infraestructura_vial.html en tu PC
2. Doble clic para abrirlo en Chrome, Edge o Firefox
3. Si ya tenes datos guardados: clic en Importar → JSON
  Necesitas internet para ver el mapa satelital
B — Cargar un nuevo registro
1. Clic en + Nuevo registro (esquina superior derecha del mapa)
2. Completar: Nombre (*obligatorio), Categoria, Ruta, Km, Localidad, Estado, Empresa
3. Coordenadas — elegir una opcion:
  · Escribir Lat/Lon decimal o GMS → el X,Y GK se calcula solo
  · Escribir X,Y GK → la Lat/Lon se calcula sola
  · Con el formulario abierto, hacer clic en el mapa para capturar la ubicacion
4. Clic en Guardar registro
C — Buscar registros
Por nombre: pestana BUSCAR, escribi en la barra o usa las letras del abecedario
Por localidad: pestana LOCALIDAD, clic en el pueblo deseado
Por categoria: en BUSCAR, filtrar por categoria en la lista lateral
Por coordenadas: pestana COORD, ingresar X,Y o Lat/Lon
D — Conversor de coordenadas (pestana COORD)
Lat/Lon → GK: escribi lat/lon (cualquier formato) y mientras escribis el X,Y GK Faja 5 se completa automaticamente. Luego "Ir al punto" mueve el mapa y muestra todos los valores.

GK → Lat/Lon: escribi X e Y y la Lat/Lon se completa sola. "Ir al punto" mueve el mapa.

Google Earth: pega el formato GMS (ej: 27°20'7.9"S) directamente en el campo Latitud — el sistema lo convierte automaticamente.
E — Exportar datos
Exportar JSON: copia de seguridad completa para Drive o pendrive. Usar para pasar datos entre PCs.

Exportar Excel: genera un archivo .xlsx con el formato oficial DVP. Aparece un selector para elegir que exportar:
  · "Todas las categorias" → exporta todos los registros
  · Elegir una categoria (ej: Gasoductos) → solo esos registros
El archivo incluye: fecha, tramite, coordenadas Lat/Lon y GK Faja 5, estado, empresa y mas.
F — Importar datos
JSON: restaura una copia completa. Reemplaza TODOS los datos actuales.

CSV / Excel: agrega registros sin borrar los existentes. El sistema detecta las columnas automaticamente. Para saber el formato correcto, descarga la plantilla Excel de ejemplo desde el modal de Importar.

La plantilla tiene dos hojas: "Registros" (para completar) y "Guia" (con categorias y estados validos).
G — Compartir la base de datos
1. Al terminar de cargar datos, clic en Exportar JSON
2. Subir el .json generado a la carpeta compartida de Google Drive de la oficina
3. Los demas agentes abren el .html → Importar → JSON → eligen el archivo de Drive
Para llevar a otra reparticion: copiar el .html + el ultimo .json en un pendrive
H — Estados de infraestructura
BUENO — Funciona normalmente. Sin danos ni riesgo.
REGULAR — Deterioro visible. Requiere mantenimiento proximo.
CRITICO — Dano importante. Requiere intervencion urgente.
SIN DATOS — Estado no verificado. Actualizar al inspeccionar.
La categoria "Otros" sirve para infraestructura que no entra en las 8 categorias predefinidas (pluviales, alcantarillas, etc). Si hay muchos del mismo tipo nuevo, crear una categoria personalizada con + Nueva categoria en la lista de categorias.
